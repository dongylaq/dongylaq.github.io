Softwares

I developed or co-developed several open source softwares. The details are listed here.

QSPPACK

QSPPACK: Quantum Signal Processing PACKage

  • QSPPACK is a Matlab toolbox for solving phase factors in quantum signal processing.

  • A set of phase factors can be used to solve various numerical quantum linar algebra problems via quantum singular value transformation.

  • Download the latest source code on Github repository.

  • Reference: Yulong Dong, Xiang Meng, K. Birgitta Whaley, Lin Lin. Efficient Phase Factor Evaluation in Quantum Signal Processing [arXiv:2002.11649].

RACBEM

RACBEM: RAndom Circuit Block Encoded Matrix

  • RACBEM is a python module implementing RACBEM using IBM's Qiskit.
  • It can be used in various quantum linear algebra problems.
  • Download the latest source code on Github repository.
  • Reference: Yulong Dong, Lin Lin. Random circuit block-encoded matrix and a proposal of quantum LINPACK benchmark [arXiv:2006.04010].
